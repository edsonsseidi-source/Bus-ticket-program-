Work Examples

Project 1: Bus Ticket Program (Python)

What I built: I created a Python program using IDLE that allows a user to select a bus destination, calculate ticket prices, and display the final cost. The program includes user input, simple validation, and basic calculations.

Tools & Technologies: Python, IDLE, variables, functions, loops, conditionals.

Skills demonstrated: Problem-solving, structuring a small program, handling user input, and applying core Python concepts.

What I learned: How to design a simple program flow, process user input, work with calculations, and debug small errors in my code. This project helped me gain confidence using Python in a practical context.

Project 2: Linux Command Practice

What I did: I practised basic Linux commands using a terminal to understand how operating systems work behind the scenes. I used commands to navigate folders, list files, check permissions, and create or delete directories.

Tools & Technologies: Linux terminal (or Windows Subsystem for Linux).

Skills demonstrated: Command-line navigation, understanding file and directory structure, and basic file permission awareness.

What I learned: How to move between directories, create and manage files, and why Linux is widely used in cybersecurity for tasks such as scripting, security tools, and server management.

Project 3: Cybersecurity Awareness Poster / Guide

What I did: I created a digital poster/guide explaining common cybersecurity threats such as phishing, malware, weak passwords, and how to stay safe online. I used course material and beginner-friendly online resources to research the topics.

Tools & Technologies: Google Docs / Word / Canva, online articles and videos.

Skills demonstrated: Researching cybersecurity topics, summarising technical information in simple language, and presenting it in a clear, visual format.

What I learned: How to identify phishing signs, why strong passwords and multi-factor authentication are important, and how to explain cybersecurity concepts to non-technical audiences.
